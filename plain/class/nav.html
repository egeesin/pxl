Navigation Component

Tip: On desktop, try to switch to mobile view from DevTools settings to observe alternative styles for the component without hover action.

Related Classes

  • -expandFrom… - All modifier classes that starts with -expand makes modified navigation component expandable. Available namespaces for this modifier class are: -expandFromBlockStart, -expandFromStartStart
  • -layout… - Modifies the layout of affected navigation component. Available namespaces: -layoutHorizontal, -layoutVertical
